# POS Keyboard Shortcuts

Quick reference for the Point of Sale screen (`/pos`). These work whenever the POS
screen is open. Function-key shortcuts are disabled while a popup (cash, card,
customer, or receipt) is open, except **Esc** and **F8**.

## Function keys

| Key | Action |
|-----|--------|
| **F2** | Focus the product **search** bar |
| **F3** | Focus the **Cash received** field |
| **F4** | **Cash** payment — opens the cash confirmation popup |
| **F6** | **Card** payment — opens the card popup |
| **F7** | Set Cash received to the **exact** total |
| **F8** | Toggle **full screen** (hides the sidebar) |
| **F9** | Open **customer** search / register |

## Cart quantity (physical numeric keypad)

| Key | Action |
|-----|--------|
| **Numpad +** | Increase the **active** item's quantity |
| **Numpad −** | Decrease the active item's quantity (removes it at 0) |

The "active" item is the one most recently added or adjusted — it is highlighted
with a blue bar on the left. Adding an item auto-scrolls the cart to it.

## Search suggestions

| Key | Action |
|-----|--------|
| **↑ / ↓** | Move through the suggestion list |
| **Enter** | Add the highlighted suggestion to the cart |
| **Esc** | Dismiss the suggestion list |

A fast barcode scan + Enter bypasses the list and adds the exact match.

## Popups (cash / card / receipt)

| Key | Action |
|-----|--------|
| **Enter** | Confirm the focused button (complete sale → print receipt) |
| **Esc** | Close the popup |
| **Backspace** | (Card popup, on the Pay button) delete a digit and return to the input |

The card popup auto-focuses the digit field; entering the 4th digit jumps to the
**Pay** button, so the flow is: `F6 → 1 2 3 4 → Enter → Enter → print`.
For cash it is: `F4 → Enter → Enter → print`.

## Product cards

| Key | Action |
|-----|--------|
| **Tab** | Move focus between product cards |
| **Enter** | Add the focused product to the cart |

## Mouse equivalents

Every shortcut has a button on screen; hover a button to see its shortcut in the
tooltip.
